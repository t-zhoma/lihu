游戏介绍

简介：该游戏由四人玩一副牌，对门一家，分别从 '3' 开始打升级，按出完手中牌的顺序计算级数，先打到 'A' 并且出户(1、2名出完牌)的一方胜出。

Online demo

http://lihu.herokuapp.com/ (Support IE9+,Chrome, Firefox, Safari)

Source Code @github

https://github.com/t-zhoma/lihu (A simple README is on the github, guys who like to download the code could find guides there)

How-To

1. Enter the game http://lihu.herokuapp.com/ (Support IE9+,Chrome, Firefox, Safari)
2. Select game room
You can see there are 10 rooms in the Select room page, each room have 4 seats, left,right,top,bottom. Click any of the seat, and on the popup dialog enter you nickname, then enter the room.
If you have not enough users to start the game (we need four users), Please select the bottom seat, so you could start the game with robots we provide :)
PS: Only the bottom seat user could control the game start
3. Waiting game room
You'll be waiting users here, and you can also talk to the users in the same room user the chat-nav-bar on the right side.
The people sit on the bottom seat could start the game.
4. Start Lihu. Just play as normal poke games. Detail rules please read this.

游戏指南

1. 游戏指南 http://lihu.herokuapp.com/ (支持 IE9+,Chrome, Firefox, Safari)
2. 选择房间
有10个房间，每个房间有四个座位，上下左右。点击任何一个座位，在弹出框中填写你的nickname，即可进入游戏。
该游戏需要四个人，如果凑不够游戏人数，请选择“下方”的座位（“下方”的座位控制游戏的开始），选择开始游戏后，系统会提供相应数量的机器人玩家以使游戏正常开始。
3. 房间等待
当进入房间，游戏尚未开始时候，你可以在房间内等待其它玩家。右侧侧边栏有简易聊天功能，可以和当前房间的其它玩家进行交流。
坐在“下”座位的玩家可以选择游戏开始
4. 开始游戏.
像一般扑克游戏一样。详细规则可以查看以下.

游戏规则

1 、发牌

一副牌 54 张，前两家各 14 张，后两家各 13 张。

2 、立户

如果当前级为 'A'， 则不用立户；反之，按出牌的顺序轮流立户。立户时可以选择 “立” 和 “不立”。立户结束后系统确定立户类型。

3 、第一个叫立户的玩家

第一轮叫立户的玩家为上一轮第一个打完牌的玩家。

4 、立户类型

不立：即 4 个玩家都不立户，
则双方4个玩家对打，由上一轮胜出的玩家首先出牌，如果一家是第1，2个出完牌就升两级；1，3个出完牌则升一级；如果1，4个出完牌，不升级，
但是下一次还是打你们的主动。反之另一家升级，升级规则一样。
单立：即只有一位玩家选择立户， 则立户玩家和两位对方玩家对打，由立户玩家首先出牌， 立户玩家第 1 个出完牌才胜出，胜出升 4 级，否则对方升 4 级；
对立：即一方的两个玩家选择立户， 则 4 个人都要打，由立户方先选择立户的玩家首先出牌，立户方第1， 2个出完牌才胜出, 胜出升 8 级，否则对方升 8 级。
斜立：即非一方的两个玩家选择立户， 则由立户的两个玩家对打，先出完牌的一方胜出，胜出方升8级。

4 、出牌

按逆时针顺序依次出牌，轮到用户跟牌时，用户可以选择 “ 不出 ” 或出比上一个玩家大的牌。

5 、牌型

单牌：单个牌（如红桃 5 ）。
对牌：数值相同的两张牌（如梅花 4+ 方块 4 ）。
双顺（滚子）：三对或更多的连续对牌（如： 334455 、7788991010JJ ）。不包括 2 点、主（当前级数）和双王。
单顺：三张或更多的连续单牌（如： 456 或 6789 ）。不包括 2 点、主和双王。
小炸弹：数值相同的三张牌（如三个 J ）。
炸弹：四张同数值牌（如四个 7 ）。
火箭：即双王（大王和小王），最大的牌。

6 、牌型的大小

火箭最大，可以打任意其他的牌。
炸弹比火箭小，比其他牌大。都是炸弹时按牌的分值比大小。
小炸弹比炸弹和火箭小，比其他牌大。都是小炸弹时按牌的分会比大小。 除火箭、炸弹和小炸弹外，其他牌必须要牌型相同且总张数相同才能比大小。
单牌按分值比大小，依次是 大王 > 小王 > 红桃主 > 非红桃主 > 红桃2 > 非红桃2 > A > ... > 3 。
